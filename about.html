About.

Color Emotion was created by Sasha Ivanov for SENG 513 (Winter 2017) Assignment 1 at the University of Calgary. Select a color to learn more about its positive and negative emotional associations. Color emotion associations gathered from coschedule.com.

Requirements for the Assignmnent:

  • The website must consist of at least 4 different pages, including a top-level page.
  • All pages must share the same style sheet.
  • It must be possible to navigate between pages:
    • it should not be possible to get stuck on any page; an easy way to accomplish this is to include at least a link to the top page on every page;
    • an even better way is to provide direct access to any page using some content navigation table/list.
  • All pages must include at least one picture.
  • The site must use (at least once) the following HTML elements:
    • table, nicely styled with CSS; and
    • list (ordered or unordered).
  • The whole site must look pretty (font, layout, color theme). Once you select a topic for a website, you can get inspiration by looking for that topic on the web.
  • Your assignment must be publicly viewable. That means you must host it on a publicly available web server. We discussed some possible hosting choices in the class.
  • All files (html & css) must be written by hand (manually). You cannot use an application to auto-generate your html or css, e.g. you cannot use MS Word to save html/css.
  • The html/css must be nicely formatted, so that a human can easily read it.
  • Make sure your site works as intended in at least the following desktop browsers: Google Chrome and Mozilla Firefox.
